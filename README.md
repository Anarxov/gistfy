## Gistfy

Gist your GitHub and Bitbucket repositories

Gistfy is a easy way to create Gists from your GitHub and Bitbucket repositories or your GitHub Gist.

## Requirements

- Node.js

## Dependencies

- express
- highlight.js
- rc
- swig

## Build dependencies

- jshint
- minify

## Install

`npm install -g gistfy`

## Usage

See the [docs](http://www.gistfy.com#api).
